# hp-top-brushup

## Overview
Homepage top page mockup/brushup for recruitment site.
Static HTML/CSS published via GitHub Pages.

## Structure
- `mockup/` -- HTML, CSS, and assets for the page mockup
- `mockup/faq-chatbot-widget/` -- FAQ chatbot widget component

## Key Files
- `mockup/index.html` -- Main page
- `mockup/style.css` -- Main stylesheet
- `mockup/job-detail.html` -- Job detail page
- `mockup/job-detail.css` -- Job detail stylesheet

## Development
- Preview: `npx serve mockup -l 8765`

## Publishing
- GitHub: https://github.com/onotaisei-create/worktalk-mockup
- Public URL: https://onotaisei-create.github.io/worktalk-mockup/
- Push procedure: Clone/pull from GitHub, copy changed files from mockup/, commit and push

## Conventions
- Pure HTML/CSS (no build step)
- Video files (.mp4, .mov) must NOT be committed to git
